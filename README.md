# Telegram Bot

Бот для Telegram с административной панелью и реферальной системой.

## Возможности

- Реферальная система
- Административная панель
- Система банов
- Рассылка сообщений
- База данных SQLite
- Управление администраторами

## Установка

1. Клонируйте репозиторий:
```bash
git clone <your-repo-url>
cd bot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Настройте переменные окружения:
- `BOT_TOKEN` - токен вашего бота от @BotFather
- `ADMIN_IDS` - ID администраторов через запятую

4. Запустите бота:
```bash
python bot.py
```

## Деплой на Apply.build

1. Загрузите проект на GitHub
2. Подключите репозиторий к Apply.build
3. Установите переменные окружения:
   - `BOT_TOKEN`
   - `ADMIN_IDS`
4. Команда запуска: `python bot.py`

## Структура

- `bot.py` - основной файл бота
- `requirements.txt` - зависимости Python
- `user_database.db` - база данных (создается автоматически)

## Команды

- `/start` - начать работу с ботом
- `/admins` - админ-панель (только для администраторов)
- `/con <unique_id>` - подключиться к пользователю
- `/msg <unique_id> <текст>` - отправить сообщение пользователю

## Безопасность

⚠️ Никогда не публикуйте токен бота в открытом доступе!
